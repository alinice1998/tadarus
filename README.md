# الحلقات الأسرية (Tadarus)

تطبيق ويب تقدمي (PWA) لإدارة حلقة تحفيظ القرآن الكريم الأسرية، يتيح لولي الأمر (المدير) والأبناء متابعة إنجازات الحفظ والمراجعة بشكل يومي.

## المميزات
- **إدارة المستخدمين:** لوحة تحكم خاصة بولي الأمر لإضافة الأبناء ومتابعتهم.
- **متابعة الإنجاز اليومي:** يمكن للأبناء تسجيل ما تم حفظه أو مراجعته.
- **نظام التحفيز والنقاط:** لوحة شرف (Leaderboard) تعرض النقاط والإنجازات لتحفيز الأبناء.
- **تطبيق ويب تقدمي (PWA):** يمكن تثبيت التطبيق على الشاشة الرئيسية للهواتف الذكية للعمل مثل التطبيقات الأصلية.
- **التوافق:** تصميم مرن يتناسب مع كافة الأجهزة (الهواتف المحمولة، الأجهزة اللوحية، أجهزة سطح المكتب).

## المتطلبات (Requirements)
- خادم ويب (Web Server) مثل Apache أو Nginx (يمكن استخدام XAMPP أو WAMP محلياً).
- إصدار PHP 7.4 فما فوق.
- قاعدة بيانات MySQL أو MariaDB.

## خطوات التثبيت (Installation)

1. **استنساخ المستودع (Clone the repo):**
   قم بتحميل ملفات المشروع ووضعها في مجلد خادم الويب (مثل `htdocs` في XAMPP).

2. **إعداد قاعدة البيانات:**
   - قم بإنشاء قاعدة بيانات جديدة (مثلاً `tadarus_db`).
   - قم باستيراد الجداول من ملف `database.sql` الموجود في المجلد الرئيسي.
     (سيتم من خلاله إنشاء الجداول وإدخال حساب المدير الافتراضي).

3. **إعدادات الاتصال (Configuration):**
   - قم بنسخ ملف `config.example.php` وأعد تسميته إلى `config.php`.
   - افتح ملف `config.php` وقم بتعديل بيانات الاتصال بقاعدة البيانات لتتطابق مع إعدادات الخادم الخاص بك:
     ```php
     define('DB_HOST', 'localhost');
     define('DB_NAME', 'tadarus_db'); // اسم قاعدة البيانات
     define('DB_USER', 'root');       // اسم المستخدم
     define('DB_PASS', '');           // كلمة المرور
     ```

   - **للشروع بالعمل:**
     - قم بإنشاء حساب ولي أمر جديد من خلال النقر على "لدي حساب" ثم "إنشاء حساب ولي أمر" في شاشة تسجيل الدخول.
